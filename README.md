# Campus-Wall_web-version-1.0
一个简单的web网页版校园墙系统，包括发布帖子、用户私聊、搜索帖子、评论点赞等功能，系统内包括交流墙、二手交易墙、好友私聊界面。

系统包括前后端两个模块，其中前端包括用户界面以及管理员界面，前后端分离。

## 网页包含的模块：

⦁	帖子发布交流区

⦁	二手交易区

⦁	求助区

⦁	好友私聊区

## 系统实现的功能：

#### ⦁	用户界面

在帖子发布交流区、二手交易区以及求助区，用户可以发布帖子实现信息交流；帖子下可以进行用户评论、点赞，发布者以及评论者可以自主删除帖子或评论；

系统支持不同用户间添加好友，并可以在好友私聊区的聊天界面进行私聊。

#### ⦁	管理员界面

管理员可以进行用户注册数据查看；

可以查看用户发布的帖子和评论，对帖子及评论内容进行审查，可以进行强制删除。

#### ⦁	后端

应用Campus后台管理系统实现，

github项目地址：https://github.com/oddfar/campus
